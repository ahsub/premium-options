3 Rollstufen (Kap. 3)
Stufe 1Niedriger Strike, gleiches Kontrakt
Bevorzugt: tieferer Basispreis, 30–60 Tage. Immer höheren Zeitwert wählen.
Stufe 2Gleicher Strike, längere Laufzeit
Wenn Stufe 1 keine positive Nettoprämie. Roll auf gleichen Strike, 30–60 Tage.
Stufe 3Niedriger Strike + Kontrakte erhöhen
Nur wenn Kapital vorhanden. Erhöht Erfolgschance drastisch — selten nötig (1–5%).

IBKR-Integration
Browser-Sicherheitsregel: GitHub Pages (HTTPS) blockiert HTTP-localhost-Verbindungen. Zwei Lösungen:
Option A — IBKR Web API (kein Proxy nötig)
IBKR HTTPS Web API direkt aus TWS. TWS → Konfiguration → API → Web API aktivieren.
Option B — Proxy mit HTTPS (einmalige Browser-Erlaubnis)
pip install flask flask-cors ib_insync pyopenssl python ibkr_options_proxy.py # startet auf https://localhost:5001 # Einmalig: https://localhost:5001 im Browser öffnen # → "Erweitert" → "Weiter zu localhost" → fertig
Finnhub API-Key (Screener)
Kostenloser Key auf finnhub.io — 60 Calls/Min. Für ~600 Aktien ca. 10–15 Min. Ergebnis wird 24h gecacht.
